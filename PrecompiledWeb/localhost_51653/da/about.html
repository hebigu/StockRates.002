Om BigumSoft

Ejet af Henrik Bigum.

Vi har udviklet

  • Biografløsning til håndtering af et næsten ubegrænset biografer
  • IT boghandel løsning
  • GPS sporing til at spore værdifulde genstande
  • Kombineret skak og chat program over nettet
  • MS SQL Server sikkerheds dokumentations løsning
  • Big Brother software til at rapportere eksekverbare programmer på en PC
  • Alarm skærm til at vise alarmer på mobile centraler
  • Kæde/hotel løsning

    • og meget meget mere

    Termer som UML2 og UP er ikke fremmed tale for os, vi værdsætter OOA og OOD, men går ikke af vejen for funktionel programmering.

    Vi er specialiseret i Microsoft SQL Server, Oracle, Sybase og Microsoft Access databaser, og udfører programmering i .NET (C# & VB.Net), men har også viden til at begå os i VBA, VBScript, Perl, PHP, Pascal og Cobol løsninger.

    Firmaet har kendskab og viden i forskellige teknologier såsom ASP.NET (herunder MVC5), Web services, WCF, WPF, ADO.NET, Entity framework, TDD, TOD etc.

    Konsulenterne kan navigere i OS som Windows til forskellige UNIX og Linux varianter.

    Firmaet har viden til at håndtere projekt Management inkluderet Agil udvikling som f.eks. SCRUM.

    Skulle de efterspørge et CMS system, kan vi også levere dette.

    Vores honorarer er rimelige, vores motto er ikke at fakturere hvis kunden ikke er er tilfreds.

    Spørg efter et tilbud, vi tror ikke de vil fortryde.
